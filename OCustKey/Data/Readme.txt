使用说明
--------
这个小工具可用来自定义智能手机的功能键，解压到手机任意目录下执行主程序即可。
因为不同手机型号支持的自定义功能键各不相同，启动程序后需要先选择手机的型号。
如果您的手机不被支持或者您发现列表中显示的功能键不正确，可自行增加或修改数据
文件，配置数据文件以 XXX.okey.xml 格式命名。

常见问题
--------
1. 修改时提示修改功能键失败。
出现该问题可能是由于手机没有解锁，工具无法创建或覆盖原来的功能键快捷方式。

2. 功能键设定无效。
由于智能手机的功能键是由键盘驱动直接调用的，即使是同一型号的手机，不同版本、
不同操作系统使用的驱动可能不同，支持的功能键快捷方式也不一样。如果发现功能键
设定无效，请查看 ROM 发布说明并修改或定制相应的配置文件。
