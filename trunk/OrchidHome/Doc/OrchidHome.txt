Orchid Home插件用于在Smartphone手机待机画面上显示农历、节假日、快捷方式等信息，
提供灵活自由的显示方式，可自定义显示内容和格式。

插件提供的功能主要有：

* 公历年月日、时分秒、星期、星座信息。
* 农历年月日、天干地支、生肖、节气等信息。
* 日出、日落、日中时间信息。
* 电量、温度、信号强度、内存、存储空间等信息。
* 公历日节假日、纪念日、生日，并支持自定义。
* 图像显示。
* 支持分组动态切换的快捷方式。
* 嵌套其它的子插件。
* 简繁体中文等多语言支持。
* 支持SP2003/2003SE/WM5系统，支持普通和QVGA屏幕。

该软件是免费软件，欢迎大家使用及提出建议！


CAB 版安装、升级和删除
----------------------
1. 安装：
将插件cab包复制到手机运行，或使用工具从PC上直接安装，安装路径选择手机。

2. 升级：
直接安装新版本的插件，或者按下面的方法先先删除旧版本后再安装新版本。

3. 删除：
先切换当前主题为不使用Orchid插件的主题，重启手机后，再到“设置”下的“删除程序”
中删除。


绿色版安装、升级和删除
----------------------
1. 安装：
1) 在PC下解压发布文件。
2) 复制 OrchidHome 目录下所有文件到手机任意目录下，如
   \Storage\Program Files\OrchidHome
3) 复制 Home 下与手机系统对应目录下的主题文件到主题文件目录下，一般是
   \Storage\Application Data\Home
4) 在手机上用资源管理器执行 OrchidHome 目录下的 OrchidRegSvr32.exe，选择
   OrchidHome.dll 并执行“操作”下的“注册”菜单项。

2. 升级：
先切换当前主题到系统主题，重启手机。再用新文件覆盖原来的文件。

3. 删除：
先切换当前主题到系统主题，重启手机。运行 OrchidRegSvr32.exe 程序，选择
OrchidHome.dll 并执行“操作”下的“卸载”菜单项，最后删除所有相关文件。


使用方法
--------
安装完成后运行手机“设置”下的“主屏幕”，在“主屏幕布局”中根据需要选择新加入的
主题。

如果使用“Orchid 快捷组”主题，该主题会自动查找开始菜单中的快捷方式，其中的分组
项分别对应以下目录：
  系统 -> Start Menu
  系统 -> Start Menu\系统
  工具 -> Start Menu\Accessories
  工具 -> Start Menu\工具
  程序 -> Start Menu\程序
  媒体 -> Start Menu\媒体
  游戏 -> Start Menu\Games
  网络 -> Start Menu\网络
只要把快捷方式分类放到以上目录中即可使用。如果需要自己定制快捷方式，参见使用手册。
